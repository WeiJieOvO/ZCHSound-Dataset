# ZCHSound: Open-Source ZJU Paediatric Heart Sound Database With Congenital Heart Disease

[[_TOC_]]

## 简介 / Introduction

ZCHSound 是一个开源的小儿心音数据库，由浙江大学医学院附属儿童医院（ZJU）建立，用于先天性心脏病（CHD）研究。

该数据库包含 **685 名儿童受试者** 的心音录音，涵盖 **正常儿童** 与 **多种先天性心脏病亚型**（如室间隔缺损 VSD、房间隔缺损 ASD、动脉导管未闭 PDA 等），总录音时长超过 **50 小时**。

论文发表于 **IEEE Transactions on Biomedical Engineering (TBME)**, 2024, Vol. 71, No. 8, pp. 2278–2286。
